CSS Properties

Listed Below are the CSS properties and their function

CSS Properties and their Functions. Click on a property ->
Propery Name Function
background-color Used to change the background color of an element like div,section,span,etc
background-image Used to apply an image as the background an element like div,section,span,etc
border Used to apply a border to an element like div,section,span,etc It takes width,type and color as argments
color Used to change the color of the text
text-align Used to change the alignement of the text ie, left,center,right or justify
box-shadow Used to apply a shadow in the x & y direction to an element
font-weight Used to change the boldness of the text
border-radius Used to change the radius of an element in order to change its angle
margin Used to set the margin of an element for any direction
display Used to apply different styles in which an element is displayed i.e, none,block,flex etc.
width Used to change the width of an element
float Used to make an element remain aligned at a position
padding Used to provide spacing to an element
